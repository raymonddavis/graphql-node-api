# graphql-node-api


## Setup
1. ``` install ```
2. Create a `.env` file.
3. Template for `.env`
```
SECRET=
DB_HOST=
DB_NAME=
DB_USER=
DB_PASS=
FACEBOOK_APP_ID=
FACEBOOK_APP_SECRET=
FACEBOOK_CALLBACK_URL=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_CALLBACK_URL=
```
4. SECRET is for crypting your JWTs, the rest should explain themselves.

## Run
```
npm start
```
